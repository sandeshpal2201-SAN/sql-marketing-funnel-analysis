# Marketing Funnel Analysis using SQL

## Objective
Analyze marketing campaign performance and identify funnel drop-offs and wasted ad spend.

## Funnel Stages
Impression → Click → Conversion

## Metrics Used
- Click Through Rate (CTR)
- Conversion Rate
- Funnel Drop-off
- Cost per Conversion

## Key Findings
- Campaign 101 converted efficiently with a cost per conversion of 7
- Campaign 102 incurred ad spend without producing any conversions
- 50% of users dropped after clicking the ad
- Post-click experience is the primary optimization opportunity

## Tools Used
- SQLite
- SQL (CASE, GROUP BY, AggregATIONS)

## Business Recommendation
Pause or optimize underperforming campaigns and improve post-click user experience to reduce wasted spend.
